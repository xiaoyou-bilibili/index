## 项目结构
- app 应用服务
- common 公共基础库
- data 存储服务
- file-front 网盘前端
- index-desktop-amis index桌面版
- index-front-amis index后台管理系统
- music-front 音乐播放器
- relation 关系服务
- search 搜索服务
- views 所有界面的json
- plugins 所有插件代码
